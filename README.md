# prime_find
C++ program/benchmark to look for primes.

# Install & Use
First clone the repo:
```
git clone https://github.com/AlexLandherr/prime_find.git
```
Change directory to repo:
```
cd prime_find/
```
Create a directory called `obj`:
```
mkdir obj
```

Then build & compile using `make`:
```
make
```

Run:
```
./prime_find
```
